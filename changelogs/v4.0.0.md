## v4.0.0 - 2026-07-06

### 新增
- **LynAI 后端联动能力增强**：完善账号、插件市场、同步和模型中转错误提示，保留后端返回的具体失败原因
- **托管模型同步兼容**：支持从 LynAI relay 配置和平铺模型列表同步 providerName、模型能力和高级参数
- **开源许可证补充**：补充随包分发、vendored、字体、前端静态资源和原生构建依赖的第三方开源组件说明，并在应用开源许可页展示

### 优化
- 同步流程在自动上传和退出前 flush 时统一记录失败状态，避免状态检查失败被静默吞掉
- 测试环境抽取 path_provider、同步服务和内存仓库辅助工具，减少真实存储副作用和重复实现
- API、账号、市场和同步服务复用统一错误解析逻辑，提升错误展示一致性

### 修复
- 修复 relay/OpenAI 风格错误 message 为空时无法回退到 type 的问题
- 修复平铺 `/relay/models` 响应中 providerName 未用于托管模型分组名的问题
- 修复 access token 为空时仍可能进入同步流程的问题
