## v2.3.1 - 2026-06-01

### 变更
- 备份 schemaVersion 升级至 3，导出格式统一对齐 storage_v2 扁平化结构
- 对话记录备份改为三数组扁平格式导出（conversations + messages + messageAttachments）
- 待办清单备份改为两数组扁平格式导出（todoLists + todoItems）
- 笔记修订备份改用扁平 delta 格式（deltaStart/deletedText/insertedText）
- 笔记修改建议备份改为 proposals 与 edit_blocks 分文件导出
- 新增 resources.json 资源注册表导出，含 SHA256 和 content-addressing 元数据
- settings.json 备份保留 storageV2.backgroundResourceId 子对象，避免 round-trip 丢失

### 新增
- 备份导入自动检测扁平/嵌套格式，兼容旧版 V1/V2 备份
- 导入时合并 resources.json 至本地资源注册表
- NoteRevision.fromJson 支持扁平 delta fallback 解析

### 修复
- 修复数据管理页未传入 StorageV2Service 导致 resources.json 导出/合并为死代码
- 修复 SHA256 匹配复用资源时 content-addressed 目标文件缺失导致引用断裂
