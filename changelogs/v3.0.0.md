## v3.0.0 - 2026-06-06

### 新增
- **Lua/WebView 插件系统**：支持安装、卸载、启用/禁用、快照备份、代码编辑器，以及工具调用和函数导出开关
- **插件配置 Schema 系统**：支持 `config.schema.json` 描述表单字段、默认值填充与校验，含模型选择器等应用专属控件
- **插件管理页面**：浏览已安装插件、查看详情、编辑 Lua 源码、管理工具/函数权限、同步内置插件
- **插件 WebView 特性页**：支持插件通过 HTML/CSS/JS 构建内嵌功能界面，集成 LynAI 全局 API
- **内置状态仪表盘插件 (status-dashboard)**：系统状态概览、资源监控、功能快捷入口
- **内置天气查询插件 (weather-query)**：通过 Open-Meteo API 查询实时天气与预报，支持中文城市名解析
- **Tree-sitter 原生结构化代码高亮**：集成 16 种编程语言的 Tree-sitter 语法解析器，通过 FFI 调用原生 C 共享库
- **代码块编辑功能**：支持在对话中点击代码块进入专用编辑器，修改后重新发送
- **代码语法服务 (CodeSyntaxService)**：统一的代码高亮与语法解析 API，Dart fallback + 原生 Tree-sitter 双路径
- **Android 系统长截图支持**：通过 Android 系统原生长截图 API 截取完整对话
- **系统滚动捕获服务 (SystemScrollCaptureService)**：跨平台长图滚动捕获抽象与实现
- **统一 AI 函数调用服务 (LynAIFunctionService)**：整合日程、笔记、待办、插件等所有 AI 可调用工具
- **模型配置选择器组件 (ModelConfigPicker)**：可复用的模型选择器，支持分组与搜索
- **插件图标组件 (PluginIcon)**：统一的插件图标渲染，支持 SVG 和默认占位图标

### 新增
- **导航双击快捷操作**：双击"功能" Tab 返回功能总览，双击"对话" Tab 打开新对话

### 优化
- 备份系统完善：支持情景演绎数据与插件数据的导出导入
- 统一插件函数 API，合并分散的工具注册逻辑
- LaTeX 渲染器全面重构，提升复杂公式渲染性能与准确性
- Mermaid 图表渲染器优化，支持更丰富的图表类型
- 移动端交互优化与消息编辑体验改进
- WebView 资源刷新与导航稳定性提升
- 插件 defaults 文件覆盖逻辑完善，支持自定义入口文件回退

### 修复
- 修复天气插件语言参数导致 HTTP 500 错误
- 修复内置插件临时目录被过早清理
- 修复长图导出渲染错位与样式丢失
- 修复内置插件源码安装与同步不一致
- 修复对话附件与 Mermaid 渲染兼容性问题
- 修复 Linux WebView 渲染稳定性问题
- 修复安卓构建配置
- 修复 CI 工作流构建与测试问题
